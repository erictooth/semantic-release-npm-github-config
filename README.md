# @erictooth/semantic-release-npm-github-config

## Getting Started
### Install
`npm i -D semantic-release @erictooth/semantic-release-npm-github-config`
### Configure package.json
**package.json**
```json
{
  "release": {
    "extends": "@erictooth/semantic-release-npm-github-config",
    "branch": "main"
  }
}
```
